Sentiment Analysis on News Using Transformers.js in Vue

July 20, 2026 Anand V

This project runs a sentiment analysis ML model completely inside the browser using Transformers.js and Vue.js. No backend, no API keys, no server costs. A Web Worker keeps the Vue UI responsive while the model processes news headlines.

1. Web Worker Setup (worker.js)

import { pipeline } from '@huggingface/transformers';

let classifier = await pipeline(
  'sentiment-analysis',
  'Xenova/distilbert-base-uncased-finetuned-sst-2-english'
);

self.addEventListener('message', async (event) => {
  let result = await classifier(event.data.text);
  self.postMessage(result);
});

2. Vue.js Implementation (App.vue)

<script setup>
import { ref, onMounted } from 'vue'

const headline = ref('')
const result = ref(null)
let worker

onMounted(() => {
  // Initialize worker
  worker = new Worker(new URL('./worker.js', import.meta.url), { type: 'module' })
  
  // Listen for results
  worker.addEventListener('message', (event) => {
    result.value = event.data[0] // e.g., { label: "POSITIVE", score: 0.99 }
  })
})

function analyze() {
  if (headline.value) {
    result.value = null
    worker.postMessage({ text: headline.value })
  }
}
</script>

<template>
  <div>
    <h2>News Sentiment Analyzer</h2>
    <input v-model="headline" placeholder="Paste a news headline..." />
    <button @click="analyze">Analyze</button>
    
    <p v-if="result">
      Result: <strong>{{ result.label }}</strong> ({{ (result.score * 100).toFixed(1) }}%)
    </p>
  </div>
</template>

Pros & Cons of this Approach

  • Advantages: Absolute privacy (data never leaves the device), zero API server costs, and it can work completely offline after the initial model download.
  • Disadvantages: The first page load can be heavy because the browser has to download the ML model (~67MB for quantized models, ~268MB without quantization). It also relies on the user's device memory and CPU, so it's not suitable for large models like GPT.

Real-World Examples

Hugging Face themselves have built working demos using Transformers.js that you can try right now:

  • Whisper Web — Real-time speech-to-text transcription running fully in the browser. You can record audio and it transcribes it locally without sending anything to a server. Try it at huggingface.co/spaces/Xenova/whisper-web
  • Browser Background Removal — Using the RMBG model by BRIA AI, developers have built tools that remove image backgrounds directly in the browser without uploading photos to any cloud server.
  • Worldline Tech Blog — Worldline, a major European payments company, published a tech blog post exploring Transformers.js for browser-side NLP tasks like sentiment analysis — showing industry interest in this approach for handling sensitive data on-device.
